下面基于 **Tauri 官方插件目录**（https://tauri.app/plugin/）给出的 **v2 系列最新插件清单**，按「桌面常用 / 移动专用 / 通用基础」三组速览——只告诉你**它是干什么的、什么时候装**，不贴大段代码，一眼就能选型。

---

### 🔧 桌面常用插件（Windows / macOS / Linux）

| 插件                 | 一句话作用                 | 典型场景               |
| -------------------- | -------------------------- | ---------------------- |
| **dialog**           | 原生文件选择、保存、消息框 | 导入导出、用户确认     |
| **fs**               | 读写文件、监听变化         | 配置文件、日志、缓存   |
| **path**             | 跨平台路径拼接、取系统目录 | 存数据库、生成下载路径 |
| **shell**            | 执行子进程、打开外部 URL   | Git 命令、打开浏览器   |
| **window**           | 多窗口、无边框、透明、置顶 | 悬浮面板、多标签       |
| **globalShortcut**   | 全局热键注册               | 老板键、截图快捷键     |
| **notification**     | 系统通知                   | 下载完成、更新提醒     |
| **clipboardManager** | 剪贴板读写                 | 复制结果、一键粘贴     |
| **http**             | 带 Cookie 的 HTTP 客户端   | 调后端、下载文件       |
| **websocket**        | 原生 WebSocket             | 低延迟行情、聊天室     |
| **updater**          | 自更新 + 签名验证          | 自动升级通道           |

---

### 📱 移动专用插件（Android / iOS）

| 插件               | 一句话作用                 | 典型场景           |
| ------------------ | -------------------------- | ------------------ |
| **barcodeScanner** | 相机扫码 / 一维码 / 二维码 | 登录、支付、加群   |
| **biometric**      | 指纹 / 面容验证            | 本地解锁、支付确认 |
| **haptics**        | 震动反馈                   | 点击、扫码成功提示 |
| **geolocation**    | GPS 定位                   | 地图、签到、运动   |
| **nfc**            | 近场通信读写               | 门禁卡、标签交互   |

---

### ⚙️ 通用基础（必装 or 常用）

| 插件              | 一句话作用               | 典型场景       |
| ----------------- | ------------------------ | -------------- |
| **api**（已内置） | 核心 IPC、事件、窗口控制 | 所有项目都依赖 |
| **os**            | 获取平台、架构、版本号   | 平台差异化逻辑 |
| **store**         | 轻量级键值持久化         | 用户偏好、缓存 |
| **stronghold**    | 加密安全存储             | 私钥、助记词   |

---

### ✅ 选型口诀

- **桌面优先**：dialog + fs + shell + window + globalShortcut
- **移动优先**：barcodeScanner + biometric + haptics + geolocation
- **通用必装**：api（内置）+ os + store / stronghold

全部插件**按需安装**，不引用就不会打包，体积最小。
